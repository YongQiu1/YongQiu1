Genshin Impact is miHoYo's first ever open-world game, where beliefs in "The Seven" converge in the fantasy world of Teyvat.

It was released for Microsoft Windows, PlayStation 4, Android, and iOS in September 2020, PlayStation 5 in April 2021, and is planned for future release on Nintendo Switch. The game features an anime style open-world environment and an action-based battle system. The game is free-to-play and is monetized through gacha game mechanics.
